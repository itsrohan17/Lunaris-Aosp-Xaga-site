Flash Instructions
⚠️ Prerequisites
✅ Unlocked bootloader
✅ PC (Windows/Linux) or secondary Android phone + OTG cable with Termux
✅ Device booted into Fastboot Mode — hold Vol Down + Power until Fastboot logo appears
❓ Understanding Installer Prompts
1. Data Wipe Prompt
[Y] Clean Flash — Formats user data. Use when coming from MIUI, HyperOS, or another ROM. ⚠️ Erases all data — back up first!
[N] Dirty Flash — Keeps data intact. Use only when updating from an older Lunaris version.
2. Boot / Root Selection
1Stock BootNo root. Best for banking apps.
2KernelSUStable kernel-level root.
3KernelSU NextCommunity fork, newer patches.
4Magisk AlphaBleeding-edge Magisk modules.
5SuKiSu UltraAdvanced KSU for power users.
Install_Lunaris.bat — Windows
1Extract the Lunaris 3.7 ZIP into a folder on your PC
2Boot K50i into Fastboot Mode and connect to PC via USB
3Double-click Install_Lunaris.bat inside the extracted folder
4Type 1 and press Enter to start the installer
5Choose your Data Wipe option (Y / N) when prompted
6Choose your Root option (1–5) when prompted
7Wait — the script flashes all partitions and reboots automatically ✅
linux_installation.sh — Linux
1Extract the Lunaris 3.7 ZIP into a folder on your PC
2Boot K50i into Fastboot Mode and connect to PC via USB
3Right-click the extracted folder → Open in Terminal
// Make script executable
$ chmod +x linux_installation.sh
// Run with sudo
$ sudo ./linux_installation.sh
6Follow on-screen menus to select Wipe and Root options
7Device reboots automatically when done ✅
install_termux.sh — Mobile to Mobile
1Install Termux on your host (secondary) Android phone
2Open Termux and install Android tools:
$ pkg install android-tools
$ termux-setup-storage
4Extract Lunaris 3.7 ZIP to host phone's internal storage
5Boot K50i into Fastboot Mode, connect to host phone via OTG cable
6In Termux, navigate to the extracted folder:
$ cd /sdcard/Download/Lunaris_Folder
$ chmod +x install_termux.sh
$ ./install_termux.sh
9Follow on-screen menus. If Termux asks USB permission → tap OK ✅
